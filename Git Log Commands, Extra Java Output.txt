GIT LOG COMMANDS

1. Git log command for Defects - Stat File = git log --stat --grep="fix" --grep="issue" --grep="closes" --grep="#" --grep="removed" --grep="correct" >defectiveFilesStat.txt

2. Git log command for Defects - One Line = git log --oneline --grep="fix" --grep="issue" --grep="closes" --grep="#" --grep="removed" --grep="correct" >defectiveFilesStat.txt

3. Git log command for No Defects - Stat File = git log --stat --grep=fix --grep=Fix --grep=Issue --grep=issue --grep=Closes --grep=closes --grep=# --grep=remove --grep=Removed --grep=Remove --grep==removed--grep=Correct --grep=correct --invert-grep > NoDefects.txt

4. Git log command for No Defects - One Line = git log --oneline --grep=fix --grep=Fix --grep=Issue --grep=issue --grep=Closes --grep=closes --grep=# --grep=remove --grep=Removed --grep=Correct --grep=correct --invert-grep > NoDefects.txt
